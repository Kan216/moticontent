ဆင်းရဲပေမယ့် မတုန်လှုပ်တဲ့ စကားလုံးများ

ထည့်သွင်းထားသော API Key များကို Browser တွင် သိမ်းဆည်းထားမည်မဟုတ်ဘဲ Server-side Session တွင်သာ လုံခြုံစွာ ယာယီသိမ်းဆည်းထားမည် ဖြစ်ပါသည်။ API key ကို aistudio.google.com/apikey တွင် အခမဲ့ရယူနိုင်ပါသည်။

ခလုတ်ကိုနှိပ်ပြီး ဒီနေ့အတွက် မတုန်လှုပ်တဲ့ စကားလုံး ၁၀ခုကို ထုတ်ကြည့်ပါ။

ထုတ်လုပ်ခဲ့ဖူးသော စကားလုံးများ မရှိသေးပါ။

သိမ်းဆည်းထားသော စကားလုံးများ မရှိသေးပါ။ ကြယ်ပုံလေးကိုနှိပ်ပြီး စကားလုံးများကို သိမ်းဆည်းပါ။
